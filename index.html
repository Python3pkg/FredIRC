Welcome to FredIRC’s Homepage!¶

FredIRC is an event-driven Python framework for IRC (Internet Relay Chat) bots. It aims to provide a high-level abstraction of IRC’s internals, an easy-to-use API and convinient utilities related to bot development.

FredIRC is developed in Python and requires at least version 3.3. It only depends on asyncio which is part of the standard library since Python 3.4 anyway.

Features¶

The main features are:

  • Easy-to-use interface for IRC bots
  • Internal event-loop that dispatches high-level IRC events
  • Different kinds of tasks which can be scheduled by the user

Note

FredIRC is already usable but the project is still in an early stage of development. So far only the very basics needed to write simple IRC bots are implemented. As long as FredIRC has a 0.x.x version number, backward-incompatible API changes might be introduced (but mentioned in the change log).

For a more detailed list of features see the Changelog.

IRC Protocol¶

FredIRC tries to implement RFC2812. But as this is no strict standard, there might be problems with specific IRC Deamons. It was mostly tested with ircd-hybrid.
